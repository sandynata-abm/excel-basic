Blok 1: Fungsi Logika Dasar (IF)

1.1 IF() - Kondisi Tunggal

Definisi: Mengembalikan nilai jika kondisi TRUE, nilai lain jika FALSE.

=IF(condition, value_if_true, value_if_false)

Contoh: =IF(D2 >= 40000000, "Target Tercapai", "Target Tidak Tercapai")

1.2 Nested IF() - Kondisi Berganda

Definisi: IF di dalam IF untuk menangani banyak kondisi bertingkat.

=IF(c1, v1, IF(c2, v2, IF(c3, v3, v4)))

Contoh: Tier komisi berdasarkan nilai penjualan.

Blok 2: Logical Operators (AND, OR, NOT)

2.1 AND()

=AND(condition1, condition2, ...)

TRUE jika semua kondisi TRUE.

2.2 OR()

=OR(condition1, condition2, ...)

TRUE jika minimal satu kondisi TRUE.

2.3 NOT()

=NOT(condition)

Membalik nilai logika TRUE/FALSE.

Blok 3: IFS() - Multi-Condition Simplified (Opsional)

Catatan Kompatibilitas:

  • IFS tersedia di Excel 2019 / Microsoft 365.
  • Excel 2013/2016 gunakan nested IF sebagai alternatif.
  • Blok ini bersifat latihan mandiri di luar kelas.
=IFS(condition1, value1, condition2, value2, ...)

Blok 4: Complex Conditions (IF + AND / IF + OR)

4.1 IF + AND

=IF(AND(c1, c2, c3), "Eligible", "Not Eligible")

4.2 IF + OR

=IF(OR(c1, c2, c3), "URGENT", "Regular")
