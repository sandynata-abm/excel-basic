Latihan Pertemuan 7

Materi: VLOOKUP dan HLOOKUP Microsoft Excel

MODUL PEMBELAJARAN

Learning Outcomes

  • C3 Apply: Menerapkan VLOOKUP, HLOOKUP, IFERROR dalam integrasi data transaksi-master
  • C4 Analyze: Menganalisis kualitas data lookup dan insight sales per kategori

Materi Pembelajaran

  • VLOOKUP() - Pencarian vertikal
  • HLOOKUP() - Pencarian horizontal
  • IFERROR() - Validasi hasil lookup
  • INDEX + MATCH (opsional mandiri)

Tips Lookup

  • Gunakan FALSE untuk exact match kode produk
  • Pakai $ (absolute reference) agar range master tidak bergeser
  • Gunakan IFERROR agar output lebih bersih
  • Approximate match (TRUE) hanya untuk tabel tier yang tersortir

LEVEL 1: 40 menit

Blok 1: VLOOKUP() - Vertical Lookup

1.1 VLOOKUP Dasar (Exact Match)

=VLOOKUP(lookup_value, table_array, col_index_num, FALSE)

Contoh: =VLOOKUP(B2, $M$2:$P$9, 2, FALSE)

1.2 VLOOKUP Multi-Kolom

Nama Produk: =VLOOKUP(B2, $M$2:$P$9, 2, FALSE)
Harga Unit: =VLOOKUP(B2, $M$2:$P$9, 3, FALSE)
Kategori: =VLOOKUP(B2, $M$2:$P$9, 4, FALSE)

1.3 VLOOKUP Approximate Match

=VLOOKUP(G2, $T$2:$U$6, 2, TRUE)

Dipakai untuk tier/range, bukan kode transaksi harian.

Blok 2: HLOOKUP() - Horizontal Lookup

2.1 HLOOKUP Dasar

=HLOOKUP(lookup_value, table_array, row_index_num, FALSE)

Contoh: =HLOOKUP("Mar", $B$20:$M$22, 2, FALSE)

2.2 Multi-Baris Parameter

Target Sales: =HLOOKUP("Apr", $B$20:$M$22, 2, FALSE)
Target Margin: =HLOOKUP("Apr", $B$20:$M$22, 3, FALSE)

Blok 3: Integrasi Lookup + IFERROR

Tujuan: Menangani kode tidak ditemukan agar output ramah pengguna.

Nama Produk: =IFERROR(VLOOKUP(B2, $M$2:$P$9, 2, FALSE), "Kode Tidak Valid")
Harga Unit: =IFERROR(VLOOKUP(B2, $M$2:$P$9, 3, FALSE), 0)

Blok 5 (Opsional): INDEX + MATCH - Belajar Mandiri

Status: Latihan mandiri di luar kelas, tidak masuk checklist penilaian utama.

=INDEX($N$2:$N$9, MATCH(B2, $M$2:$M$9, 0))
=IFERROR(INDEX($O$2:$O$9, MATCH(B2, $M$2:$M$9, 0)), 0)
