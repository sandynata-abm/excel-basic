Tips Praktik

Gunakan tombol Ctrl + S sesering mungkin untuk menyimpan progres latihan Anda di Excel agar data tidak hilang.

Panduan Dasar

Ribbon & Backstage

Gunakan tab File untuk akses Save, Print, dan New (Backstage View).

Cell Basics

Cell adalah pertemuan baris dan kolom (A1, B5). Gunakan Backspace untuk menghapus isi.

Formatting

Ubah tampilan teks di tab Home (Bold, Color, Merge & Center, Borders).

Number & Date Format

Atur tipe data (Currency, Short/Long Date, Percentage) pada grup Number di tab Home.
